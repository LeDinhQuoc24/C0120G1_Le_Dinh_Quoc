Các phương thức của đối tượng Date getDate(): Trả về ngày trong tháng (từ 1-31) getDay(): Trả về ngày trong tuần (từ 0-6) getFullYear(): Trả về năm getHours(): Trả về giờ (từ 0-23) getMinutes(): Trả về phút (từ 0-59) getSeconds(): Trả về giây (từ 0-59) getMilliseconds(): Trả về phần nghìn giây (từ 0-999) getTime(): Trả về tổng số milliseconds (tính từ nửa đêm 1/1/1970)
